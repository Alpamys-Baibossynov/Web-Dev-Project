# Web-Dev-Project
Watchlist project for Web Development discipline

1. Baibossynov Alpamys
2. Zhumamurat Bekzat

We will create a full-stack web application that allows
users to manage their personal list of movies, animations,
TV shows, series, etc.

Our project is inspired by Letterboxd, Serializd, MyAnimeList applications.
Big disadvantage of these apps that they specialized on one section of content.
We want to create an application to track all stuff in one place.
